Wispr Clone

Voice typing for your browser.

How to use

  1. Click into any text box on a web page (Gmail, X, ChatGPT, a search bar…).
  2. Press ⌘⇧Y (or click the toolbar icon → Start).
  3. The first time on each website, Chrome asks for the microphone — click Allow.
  4. Speak. Your words are typed into the field. Press the shortcut again to stop.
💡 Just installed? If you press the shortcut on a tab that was already open, reload that tab once so the extension can attach to it.

Troubleshooting

  1. Nothing happens: reload the page, then try again.
  2. "Allow microphone" message: click the 🎤/lock icon in the address bar and allow the mic for that site.
  3. Only works on secure sites: speech needs https:// pages (most sites are).

Note: in-browser dictation uses the browser's built-in speech engine (Google), so audio is processed in the cloud. For fully on-device, private dictation, use the Wispr Clone Mac app.
