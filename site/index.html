Prefer your browser? 🧩

Get the free Chrome extension and dictate straight into Gmail, X, ChatGPT, or any web text box — no Mac app needed. Works on Windows & Linux too.

Get the Chrome extension
Free · Chrome / Edge / Brave
🧩 How to install
  1. Unzip the download, then open chrome://extensions in Chrome.
  2. Turn on Developer mode (top-right), then click Load unpacked.
  3. Select the unzipped extension folder.
  4. Click the toolbar icon ▸ Enable microphone (one time), then press ⌘⇧Y in any text field and speak.

Browser dictation uses the browser's built-in speech engine (cloud). For fully on-device, private dictation, use the Mac app above.

📋 How to install (first time)
  1. Open the downloaded Wispr-Clone.dmg and drag Wispr Clone into your Applications folder.
  2. Open Applications and double-click Wispr Clone. macOS will block it — that's expected.
  3. Go to System Settings ▸ Privacy & Security, scroll down, and click “Open Anyway” next to Wispr Clone. Confirm once.
  4. When it runs, allow Microphone, Speech Recognition, and Accessibility when asked, then relaunch.

Why the warning? This app isn't notarized by Apple yet, so macOS asks you to approve it manually. It's safe — the code is open source on GitHub.
